<div align="center"> 
  <h3>이버스마켓이란? (e-verse-market)</h3>
  "e"는 전자(Electronic), "Verse"는 여러 가지를 포함하는 것이라는 뜻으로, <br />
  개인이나 기업이 디지털 플래너, 발표자료(PPT 양식), 디지털 스티커 등등 <br />
  다양한 디지털 상품들을 판매할 수 있도록 도와주는 플랫폼이다.  <br/>
</div>
